Blogify!

In this project I created a multi user blog(along the lines of Medium) where users can sign in and post blog posts as well as 'Like' and 'Comment' on other posts made on the blog. The blog is hosted on Google App Engine and uses a personally created authentication system for users to be able to register and sign in and then create blog posts!
Tech used: Google App Engine, HTML&CSS, JS, and Python.
